# GoalCraft

GoalCraft is a teacher-focused IEP goal generator built to make goal writing faster, clearer, and more reusable.

## Features
- Domain-based goal builder
- Goal Bank with search and filtering
- Favorites saved in local storage
- Goal quality and compliance checks
- Paragraph and structured goal formatting
- Copy and .txt export

## Privacy
Goal favorites are stored locally in the user’s browser. The app does not intentionally store saved goals in a remote database.

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
